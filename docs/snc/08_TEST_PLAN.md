## 3. 建议的测试用例

### 测速引擎
- 正常延迟值（50ms, 200ms, 1000ms）
- 超时场景（>5000ms）
- 连接失败场景
- 评分权重为 0 的边界情况

### 解析器
- 标准 Base64 编码
- URL-safe Base64
- 空内容、无效内容
- 多种协议混合（ss://, vmess://, vless://）
- YAML 格式（Clash proxies 列表）
- CSV 带特殊字符

### 去重
- 完全相同节点
- 相同地址不同端口
- 相同端口不同协议
- 大量节点（1000+）性能
